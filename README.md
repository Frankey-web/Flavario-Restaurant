# Flavorio – Premium Restaurant Website

Flavorio is a modern, fully responsive restaurant website designed to deliver a premium dining experience online. It features elegant UI/UX, smooth animations, interactive food galleries, table reservations, Google Maps integration, and a mobile-first design that ensures a seamless experience across all devices.

Built with clean, well-structured HTML, CSS, and JavaScript, this project demonstrates attention to detail, performance, responsiveness, and maintainable code. It reflects my ability to create visually appealing, user-focused websites tailored for real businesses.

Key Features

- Modern and elegant restaurant interface
- Fully responsive across desktop, tablet, and mobile
- Interactive menu and gallery sections
- Table reservation system
- Google Maps integration
- Optimized performance and clean code structure
- Professional UI/UX with smooth user interactions
